## O que é Lógica
A lógica de programação, por sua vez, é a habilidade de pensar e estruturar a solução de um problema de forma lógica, utilizando conceitos como sequência, seleção (decisão) e repetição. Ela é fundamental para desenvolver algoritmos eficientes e eficazes.

Para desenvolver uma lógica de programação sólida, é necessário entender os conceitos básicos, como:
    • Sequência: Determinar a ordem correta das instruções para executar uma tarefa em uma sequência lógica.
    • 
    • Seleção (Decisão): Fazer escolhas com base em condições específicas. Por exemplo, executar um bloco de código se uma determinada condição for verdadeira e outro bloco se for falsa.
    • 
    • Repetição: Executar um conjunto de instruções repetidamente enquanto uma condição for verdadeira ou por um número específico de vezes.
    • 
    • Variáveis: Armazenar e manipular valores que podem ser usados ​​em diferentes partes do programa.
    • 
    • Estruturas de dados: Utilizar estruturas como arrays e objetos para armazenar e organizar conjuntos de dados.
    • 
    • Funções: Agrupar um conjunto de instruções em um bloco reutilizável, que pode ser chamado várias vezes em diferentes partes do programa.
